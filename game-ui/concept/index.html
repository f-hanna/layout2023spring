Description

Basically will utilize the setting of a JRPG, probably taking the most inspiration from Octopath Traveller (GOAT game). The player will have the ability to either attack, use their character skill, defend, pick an item, or flee. Both attacking and character require the use of a mana bar. Picking an item requires an inventory.
